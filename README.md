# ✨ vibe

> ✨ Scaffold projects at the speed of thought. Minimalist, zero-config starter kits.

## Features
- **🎨 Beautiful TUI**: Built with `Typer` and `Rich` for a stunning visual experience.
- **⚡ Zero Config**: Get directly into the flow state without writing boilerplate.
- **🚀 Instant Starters**: Generate React, FastAPI, or Node templates instantly.

## Quick Start
```bash
pip install vibe-cli
vibe create my-project --template react
```
